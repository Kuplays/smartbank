<h1>Реестр транзакций</h1>

![Example](/res01.png)

Простой реестр транзакций клиентов, без добавления параллельной обработки

* Технологии:
  * Сервер:
    * Java Spring API
    * PostgreSQL
  * Клиент:
    * TypeScript
    * AngularJS
    * Bootstrap
